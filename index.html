Импорт отдельного конфига вручную

Если твой клиент не поддерживает подписки (или нужен только один сервер), используй ручной импорт:

  1. Найди нужный сервер в списке ниже (по стране)
  2. Нажми 📋 (копировать) рядом с сервером
  3. Открой свой VPN-клиент
  4. Нажми "+" → "Import"
  5. Готово — можно подключаться
Импорт через QR-код

Для мобильных клиентов удобно использовать QR-код:

  1. Найди нужный сервер в списке ниже (по стране)
  2. Нажми 📲 (QR) рядом с сервером
  3. Откроется окно с QR-кодом
  4. Открой VPN-клиент на телефоне
  5. Выбери "Scan QR" или "Сканировать"
  6. Наведи камеру на экран — конфиг добавится автоматически
